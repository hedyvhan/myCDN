# 第一个程序

> 2021-05-19

1. 编写代码
   1.  文件名为hello_world.java
   2. 代码的类名必须和文件名相同
   
2. javac编译：写完代码，进入当前目录下，使用javac hello_world.java 生成.class文件

3. java执行：使用java hello_world运行这个.class文件。

    如果有包，需要在包名目录下， 并带上hello_world的路径。
